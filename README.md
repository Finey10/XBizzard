# XBizzard
🎂 Cyberpunk Birthday Archive

An interactive cyberpunk-themed birthday website created as a digital birthday gift for one of my closest friends. Since I couldn't celebrate in person, I wanted to create something more meaningful than a simple message—a website that preserves our memories in a futuristic "memory archive."

The experience takes the user through different chapters of our friendship, from childhood memories to recent moments, using cinematic animations, glitch effects, neon aesthetics, and interactive galleries.

✨ Features
🌆 Cyberpunk-inspired UI with neon pink & cyan aesthetics
📸 Interactive photo timeline and memory archive
🎥 Video gallery with custom playback interface
⚡ Smooth scroll animations and glitch transitions
🎂 Personalized birthday message and cinematic ending
🖼️ Responsive design for desktop and mobile
🎮 Hidden easter eggs and interactive elements
💾 Digital "Memory Archive" storytelling experience

❤️ Why I Built This

Sometimes distance makes it impossible to celebrate special moments together. This project is my way of turning years of memories into an interactive experience—a small digital time capsule dedicated to a friend who has been part of many unforgettable moments.
